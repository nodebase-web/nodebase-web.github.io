⭐ Причины для вноса

1Главные (один раз)

Кредит на номера (МТС / Теле2 / Мегафон), распространение вирусов (инжекторы, читы), взлом ТГ / Роблокс, фейк гаранта / админа / владельца / бота.

1+Суммируемые (каждый раз +1)

Скам на Adopt Me / MM2 / PSX / Сонарию / Арты / Позинг / Канал / Валюту / Робуксы / Коды / Звёзды / ТГ / Роблокс аккаунт. Взлом-ссылка / Фишинг.

0Информационные (без счёта)

Попытка скама, отказ от гаранта (прямой / свой НН-гарант / повышение цены), сомнительные услуги (донат через пароль / траст-трейд / аренда / прокачка / фриланс / понижение цены), фейк чека/пруфов/инвентаря, украденные пруфы/инвентарь, накрученные пруфы, сотрудничество в скаме/взломе / реклама скам-бота, диз-информация, просьба/обучение взлому, попытка ухода с базы (твинк).

📜 Правила чатов

💬 Оффтоп

Свободное общение. ❌ ЦП, расчленёнка, слив данных — бан. Спам (≥10 за 5 с) — мут.

📥 Предложка

Чат для отправки информации о скамерах. Закрывается на ночь. Оффтоп → варн, 2 варна → мут 24 ч. Вызов админа: /report причина.

⚖️ Суд

Обжалование добавления в базу. Уважение к судьям, только по делу, полные доказательства. ❌ 18+.

⚠️ Наказания

Варн — предупреждение · Мут — ограничение · Бан — блокировка.

📘 Для администрации

🟢 Начало смены

  • «беру сит» в ответ на последний закреп
  • Если в закрепе доразобрать: ссылка — перейти и разбирать оттуда
  • Строго сверху вниз, на каждый сит — «беру»

🔚 Завершение смены

  • /offline всё разобрано после разбора
  • При перерыве ≥10 мин — указать следующий сит
  • /night (закрыть), /day (открыть)

🔍 Разбор ситуации

  • Скрин сообщения + профиля отправителя (с ID)
  • При соответствии — пруфы в канал доказательств (скрины + подпись с ID)
  • Внос: /add ID [кол-во] [причина] [ссылка]

🛡️ Модерация

  • /warn за нарушение (ответом на сообщение)
  • 2 варна → автоматический мут
  • Репорт: перейти по ссылке → /warn → «Просмотрено»
Правило 2 админов: в предложке одновременно не более 2 админов во избежание путаницы.

📋 Категории причин

1 +1 (один раз)

  • Кредит на номера — МТС / Теле2 / Мегафон
  • Вирусы — инжекторы, читы, вредоносные программы
  • Взлом — ТГ аккаунта
  • Взлом — Роблокс аккаунта
  • Фейк — гаранта / бывшего / скам-гаранта / админа / владельца / бота

1+ +1 (каждый раз)

  • Скам на: Adopt Me / MM2 / PSX / Сонарию / Арты / Позинг / Канал / Валюту / Робуксы / Коды / Звёзды / ТГ / Роблокс
  • Взлом-ссылка / Фишинг — скрытые ссылки, QR-коды

0 +0 (информационные)

  • Попытка скама — на всё из 1+ / фото из сети / через фейк гаранта
  • Отказ от гаранта — прямой / повышение цены / свой НН-гарант
  • Сомнительные услуги — донат через пароль / траст-трейд / аренда ТГ/ВК/ВЦ/MAX / прокачка / фриланс / понижение цены на товары
  • Фейк — чека, пруфов, инвентаря
  • Украденные — пруфы, инвентарь
  • Накрученные пруфы — покупка отзывов, замазанные ники, перескоки нумерации
  • Обучение взлому — просьба / обучение
  • Сотрудничество — в скаме / во взломе / реклама скам-бота
  • Диз-информация — на админа / гаранта / владельца

📐 Требования к пруфам

  • Скрины полные, без обрезок и замазок
  • Полная переписка, факт сделки (скрин трейда / чек)
  • Финал: блок / игнор от 3 дней (или 2 нед. обещаний)
  • ГС: пересыл или видео с экрана. Профиль скамера виден
  • Накрутка: перескоки нумерации, замазанные ники (≥10), отзывы на игровые валюты (≥5), сделки до 10₽ (≥10), просьба написать отзыв не на его профиль доверия
  • Домены: только roblox.com / www.roblox.com / web.roblox.com — остальное фишинг
  • Фишинг: /scam_link + /add «Взлом ссылка»

⚠️ Наказания АБ

  • Ошибка при добавлении → админ-варн
  • Испыту: макс. 2 варна (при втором — снятие)
  • Админ: макс. 5 варнов (+1 за 1000 добавлений)

🎯 Испытательный срок

  • 50 ситов за 3 дня
  • Лимиты: Фейки — 15, Отказ от гаранта — 15, Сомнительные услуги — 10, Взлом — 10
  • Ситы сверх лимита не учитываются
